Takes a set of numbers and calculates their collective quotient.

#### An Example

```scheme
(/ 10 3)      ;; returns 3 (two integers)
(/ 10 3.0)    ;; returns 3.3333333333333335
(/ 20 2.0 4)  ;; returns 2.5
```
